# upmon-tray
The upmon tray application.

## Configure

* Create an `.upmonrc` file (see https://github.com/alanshaw/upmon#getting-started)
* Start the app

## Build

### OSX x64

```sh
npm run build:darwin
```

### Linux x64

```sh
npm run build:linux
```

## run

```sh
npm start
```
